1. Округление оценок
Вы можете выставлять «частичные баллы» за ответы на экзаменационные вопросы, но итоговая оценка за экзамен должна быть равна ints. Поэтому, прежде чем что-то делать с оценками в классе, нужно просмотреть их и преобразовать все float оценки в ints. К счастью для вас, в Python есть встроенная функция round(), которой вы можете воспользоваться.

Создайте функцию round_scores(student_scores), которая принимает list из student_scores. Эта функция должна принимать входные list и return новый список со всеми оценками, преобразованными в ints. Порядок оценок в результирующем list не важен.


2. Студенты, не сдавшие экзамен
Во время проверки экзамена вы заметили, что некоторые студенты справились не так хорошо, как вы надеялись. Но вы отвлеклись и забыли записать, сколько именно студентов не сдали экзамен.

Создайте функцию count_failed_students(student_scores), которая принимает list из student_scores. Эта функция должна подсчитать количество студентов, не набравших проходной балл, и вернуть это значение в виде целого числа. Чтобы получить проходной балл на экзамене, студент должен набрать больше 40 баллов.


3. «Лучшие»
Учитель, которому вы помогаете, хочет найти группу учеников, показавших «лучшие» результаты на этом экзамене. Критерии «лучших» результатов постоянно меняются, поэтому вам нужно найти учеников, набравших больше или равное количество баллов.

Создайте функцию above_threshold(student_scores, threshold), принимающую student_scores (list оценок) и threshold (пороговый показатель «максимального балла») в качестве параметров. Эта функция должна возвращать list всех оценок в диапазоне от >= до threshold.


4. Выставление буквенных оценок
Учитель, которому вы помогаете, любит выставлять буквенные оценки в дополнение к числовым баллам. Поскольку ученики редко набирают 100 баллов на экзамене, нижние пороговые значения для «буквенной оценки» рассчитываются на основе наивысшего набранного балла и равномерно распределяются между наивысшим баллом и пороговым значением <= 40, при котором оценка считается неудовлетворительной.

Создайте функцию letter_grades(highest), которая принимает в качестве аргумента «наивысший» балл за экзамен и возвращает list более низких пороговых значений для каждого интервала оценок в «американском стиле»: ["D", "C", "B", "A"].


5. Сопоставление имен с баллами
У вас есть список оценок за экзамен в порядке убывания, а также список имен учащихся, также отсортированный в порядке убывания по оценкам за экзамен. Вам нужно сопоставить имя каждого учащегося с его оценкой за экзамен и вывести общий рейтинг класса.

Создайте функцию student_ranking(student_scores, student_names) с параметрами student_scores и student_names. Сопоставьте имя каждого студента в списке student_names list с его оценкой в student_sores list. Можно предположить, что каждый аргумент list будет отсортирован от наивысшего балла (er) к наименьшему баллу (er). Функция должна возвращать list строки с форматом <rank>. <student name>: <student score>.


6. «Идеальный» результат
Хотя «идеальный» результат в 100 баллов на экзамене встречается редко, интересно узнать, был ли он у кого-то из студентов.

Создайте функцию perfect_score(student_info) с параметром student_info. student_info — это list списков, содержащих имя и оценку каждого ученика: [["Charles", 90], ["Tony", 80]]. Функция должна возвращать return первую [<name>, <score>] пару значений для ученика, набравшего 100 баллов на экзамене.

Если в student_info не найдено ни одного результата со счетом 100, должен быть возвращен пустой список [].

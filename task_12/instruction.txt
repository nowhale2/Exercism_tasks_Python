1. Извлечь координаты
Реализуйте функцию get_coordinate(), которая принимает пару (treasure, coordinate) из списка Azara и возвращает только извлеченные координаты на карте.


2. Форматирование координат
Реализуйте функцию convert_coordinate(), которая принимает координату в формате «2A» и возвращает кортеж в формате ("2", "A").


3. Сопоставьте координаты
Реализуйте функцию compare_records(), которая принимает (treasure, coordinate) пару и (location, coordinate, quadrant) запись и сравнивает координаты в каждой из них. Возвращайте True в случае совпадения координат и False в случае несовпадения. При необходимости переформатируйте координаты для точного сравнения.


4. Объедините совпадающие записи
Реализуйте функцию create_record(), которая принимает (treasure, coordinate) пару из списка Азары и (location, coordinate, quadrant) запись из списка Руи и возвращает (treasure, coordinate, location, coordinate, quadrant) если координаты совпадают. Если координаты не совпадают, верните строку «не совпадают». При необходимости переформатируйте координаты для точного сравнения.


5. «Очистка» и составление отчета по всем записям
Очистите объединенные записи от Azara и Rui, чтобы в каждой записи был только один набор координат. Составьте отчет, чтобы они могли видеть единый список всего, что нужно нанести на карту. Реализуйте функцию clean_up(), которая принимает кортеж кортежей (все из обоих списков), перебирает внешний кортеж, удаляет ненужные координаты из каждого внутреннего кортежа и добавляет их в «отчет». Отформатируйте и верните 'отчет' так, чтобы в каждой строке была одна очищенная запись.
